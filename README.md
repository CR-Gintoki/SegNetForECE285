### Packages

Packages that need installation: numpy, Pillow, matplotlib, torch, torchvision.

Use the following commands to install packages.

Install numpy:
```Shell
$ pip install numpy
```

Install Pillow:
```Shell
$ pip install Pillow
```

Install matplotlib: 
```Shell
$ pip install matplotlib
```

Install torch: 
```Shell
$ pip install torch
```

Install torchvision: 
```Shell
$ pip install torchvision
```
